# Ochuunn


## Install
1. In the `root` directory:

    ```git
    $ git clone https://github.com/ochukai/hexo-theme-ochuunn.git themes/ochuunn
    ```

2. Change the `theme` property in the `config.yml` file.

    ```yml
    # theme: landscape
    theme: ochuunn
    ```

3. Run: `hexo generate` and `hexo server`
